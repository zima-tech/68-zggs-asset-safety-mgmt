# 资产安全管理

## 原需求描述

资产安全全过程留痕，分析租户安全风险，提供数据支撑

---

## 页面结构

```
- 安全管理首页
  - 安全态势
  - 预警提示

- 租户管理页
  - 租户档案
  - 风险评估

- 访客管理页
  - 访客登记
  - 异常记录

- 安全巡检页
  - 巡检任务
  - 记录查询
```

## 功能列表

### 1. 租户安全档案

- 租户信息：
  ```
  [租户档案]

  租户：张三
  房间：XX公寓302
  入住时间：2024-01-15
  安全风险等级：低

  档案记录：
  ├─ 入住审核：[通过]
  ├─ 违规记录：[无]
  └─ 缴费记录：[正常]
  ```

- 安全风险评估
- 历史记录追踪

### 2. 租户风险分析

- 风险评估模型
- 风险预警：
  ```
  [风险预警]

  租户：李四
  风险等级：中等

  风险因素：
  ├─ 多次逾期缴费
  ├─ 涉嫌转租
  └─ 群租嫌疑

  [建议] 实地核查
  ```

### 3. 全过程留痕

- 操作日志记录
- 证据链留存
- 审计追溯

### 4. 访客管理

- 人脸识别登记
- 来访记录：
  ```
  [访客记录]

  访客：王五
  被访人：张三
  来访时间：2024-01-15 14:00
  离开时间：2024-01-15 16:30
  访问目的：探访

  [人脸核验通过]
  ```

### 5. 数据支撑

- 统计分析报表
- 决策支持数据

## 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| tenant_id | string | 租客ID |
| risk_level | enum | 风险等级 |
| risk_factors | array | 风险因素 |
| visit_records | array | 来访记录 |
| operation_logs | array | 操作日志 |
